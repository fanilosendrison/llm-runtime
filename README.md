# llm-runtime

Runtime TypeScript pour orchestration de providers LLM (completion + embedding) — librairie Node >= 20.

## Stack

- **Language** : TypeScript 5.8 (ESM strict)
- **Runtime** : Node 22 LTS (plancher >= 20)
- **Package manager** : pnpm
- **Test runner** : vitest
- **Linter** : biome
- **Type checker** : tsc

## Status

WIP — initial scaffolding. Specs normatives dans `specs/`.

## Getting started

```bash
pnpm install
pnpm test
pnpm build
```
